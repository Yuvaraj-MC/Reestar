# 🍽️ ReeStar - Food Website

ReeStar is a simple and visually appealing food website developed using **HTML5** and **CSS3**. This project showcases a collection of delicious food items in a structured card layout along with a basic contact form.

## 🚀 Features

* 📌 Clean and modern user interface
* 🍜 Food items displayed using card layout
* 🎨 Styled using CSS Flexbox
* 📱 Basic responsive design
* 📩 Contact form for user interaction

## 🛠️ Technologies Used

* HTML5 (Semantic Elements)
* CSS3 (Flexbox, Styling)

## 📂 Project Structure

```
ReeStar/
│── index.html
│── index.css
│── images/
```

## 🎯 Learning Outcomes

* Understanding of semantic HTML structure
* Hands-on experience with CSS layout techniques (Flexbox)
* Building responsive UI components
* Designing a simple multi-section webpage
